## Goal

gello as a daily tool across repos: open any project, initialize a board in a
fresh repo, archive old cards, and ship a signed installable build.

## Definition of done

- Folder picker + recent-projects list; opening a repo without `.gello/`
  offers to scaffold one (including the CLAUDE.md convention snippet).
- Done cards can be archived to `archive/` without breaking links.
- A distributable macOS build exists.
